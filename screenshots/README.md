This directory stores screenshots of the Streamlit UI for the README.

## How to take screenshots

1. Run the app: `streamlit run app.py`
2. Open http://localhost:8501
3. Take a screenshot of:
   - The home/upload screen → save as `home.png`
   - A detection result → save as `prediction.png`
4. Update the table in README.md to point to these files.

## Recommended screenshot dimensions

- Width: 1280px or wider
- Format: PNG (lossless, looks sharper in GitHub)

These files are intentionally not committed as placeholders.
Add real screenshots before publishing the repository.
